## 📖 О проекте

**«Анти ИИ Qwen БПЛА Тренажёр»** — это бесплатный социальный офлайн-тренажёр, разработанный в рамках образовательной платформы 
**Anti-AI Qwen Academy**. Проект предназначен для первичной подготовки операторов беспилотных летательных аппаратов (БПЛА)
и развития базовых навыков управления дроном в виртуальной среде.

### 🎯 Цель проекта
Дать возможность любому желающему **бесплатно и без интернета** попробовать себя в роли оператора БПЛА, отработать 
навыки пилотирования и прицеливания в безопасной виртуальной среде.

### 🌟 Особенности
- ✅ **Полностью офлайн** — работает без интернета (кроме первой загрузки Three.js)
- ✅ **Бесплатно навсегда** — социальный проект, без рекламы и донатов
- ✅ **Один HTML-файл** — не требует установки, просто откройте в браузере
- ✅ **Адаптивность** — работает на слабых устройствах (автоопределение)
- ✅ **Физика столкновений** — дрон реально падает при сильных ударах
- ✅ **Русская раскладка** — поддержка WASD и ЦФЫВ

## 🎮 Геймплей — Урок 1: Тир

В первом уроке оператор «Диджей» отрабатывает:
- 🎯 **Пилотирование** — удержание дрона в воздухе, маневрирование
- 💥 **Прицеливание** — наведение на движущиеся мишени
- 🛡️ **Избегание столкновений** — контроль дистанции до стен
- ⚡ **Стабилизацию** — восстановление контроля после манёвров

**Задача:** поразить 5 движущихся мишеней, не разбив дрон о стены.

---

## 🕹️ Управление

| Клавиша | Действие |
|---------|----------|
| `W` / `Ц` / `↑` | Движение вперёд |
| `S` / `Ы` / `↓` | Движение назад |
| `A` / `Ф` / `←` | Движение влево |
| `D` / `В` / `→` | Движение вправо |
| `Пробел` | Стабилизация дрона |
| `ЛКМ` | ЭМИ-удар (выстрел) |

> ⚠️ **Важно:** прицел находится в центре экрана — наводите дрон так, чтобы мишень была в перекрестье!
